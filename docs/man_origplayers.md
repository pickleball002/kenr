all other fields are zero.  

KennyZ,4.0,0,0,0,0.00000,0.00000,4.00000,m

current 'kenr' rating.   In this file it doesn't change unless your reboot the ratings
games played
games won
games lost
winning percentage
how much above original 'kenr' you are
original 'kenr'.
gender

future files like 'players' or 'deviation' reflect changes.

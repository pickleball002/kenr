The 'players' file is in the /prod directory.   This is the file which gets changed whether you use batch or interactive mode.

It is sorted by highest 'kenr' rating, column 2.

If everyone starts with the same 'kenr' rating, then this could be a good sort.

I call this the 'top level' rating.   All games are in here, including games that are not
necessarily mens/womens/mixed.

Let's take a look at some sample output from testing Spring 1 session of rec play.

TestPlayer,4.07016,87,48,39,0.55172,0.07016,4.00000,m

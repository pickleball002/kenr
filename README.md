# kenr
KENR is an enterprise pickleball ranking system.   Built with ChatGPT and using Perl I created a ranking idea for pickleball players.   Makes it easy to self host your own Perl scripts.   Created in response to current ranking systems that everyone seems to complain about.

Any questions you can go to 'Discussions'.

Version 1.   One top level rating, everyone is 4.0
Version 2.   Adding new fields, gender and ratings.   Beginnings of different levels.
Version 3.   Batch processing added in addition to initial interactive.

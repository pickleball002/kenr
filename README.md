# kenr
KENR is an enterprise pickleball ranking system. \
Built with ChatGPT and using Perl I created a ranking idea for pickleball players.  

Makes it easy to self host your own Perl scripts.  \
Created in response to current ranking systems that everyone seems to talk about.

Of course it's free for anyone to use.   Great for groups that want to avoid a current rating system, or just keep track of
wins and losses.

Any questions you can go to 'Discussions'.

# kenr
KENR is an enterprise pickleball ranking system.   Built with ChatGPT and using Perl I created a ranking idea for pickleball players.  
Makes it easy to self host your own Perl scripts.   Created in response to current ranking systems that everyone seems to complain about.

I am building up the 'docs' folder. 
If you create the 'origplayers' file (or ChatGPT builds it) then if results are formatted well It's easy for me to run.

Any questions you can go to 'Discussions'.
